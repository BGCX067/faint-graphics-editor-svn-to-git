= Replace Color =
Replace a color in the image with a different color.

== Method ==
Replacing color is only available from the Python interpreter.

*Image methods*
|| image.replace_color(old_color, new_color) || Replaces old_color with new_color ||

|| image:icon_info.png || *Tip:* Use the -tool_picker,color picker- to select the color to delete, and use 'get_fgcol()' or 'get_bgcol()' as an argument to 'replace_color'. ||
