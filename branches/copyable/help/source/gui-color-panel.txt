= The color panel =
The color panel contains a set of configured colors, gradients and
patterns.

\image(color_palette.png)

||\image(mouse_left.png)|| Left-click to select a color as primary  ||
||\image(mouse_right.png)|| Right-click to select a color as secondary ||
||\image(mouse_left_double.png)|| Double-click to open the \ref(gui-color-selection-dialog.txt) ||
||\image(key_ctrl_mouse_right.png)|| Ctrl+Right-click to remove a color from the panel ||

The colors can be reorganized by drag and drop.
